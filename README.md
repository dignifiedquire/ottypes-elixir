# Ottypes

[![Build Status](https://travis-ci.org/Dignifiedquire/ottypes-elixir.svg)](https://travis-ci.org/Dignifiedquire/ottypes-elixir)

> An implementation of different ottypes in Elixir.


Available types:

* `src/text.ex`



## Development

Building

```bash
$ mix compile
```

Running tests:

```bash
$ mix test
```

Running dialyzer:

```bash
$ mix dialyze
```
